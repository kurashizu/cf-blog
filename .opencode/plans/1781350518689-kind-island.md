# Plan: Split HN Fetching from Summarization + Heartbeat Cron

## Overview

- 30-min cron: fetch top 5 HN stories, store in D1 with empty `summary`
- New 3-min cron (`heartbeat`): pick one `news_items` with empty summary, fetch article content, call Gemini for detailed summary, update D1
- Individual per-item summarization (not batch), longer summaries covering article content

### 6. `lib/heartbeat.ts` **New**

```typescript
export async function handleHeartbeat(env: Env): Promise<{ ok: boolean; detail: string }> {
    // 1. Find one item with empty summary
    const row = await env.DB.prepare(
        "SELECT * FROM news_items WHERE summary = '' ORDER BY time ASC LIMIT 1"
    ).first();

    if (!row) return { ok: true, detail: "nothing pending" };

    // 2. Generate summary
    const story = row as HNStory;
    const summary = await generateSingleSummary(story, env.GEMINI_API_KEY!);

    // 3. Update D1
    await env.DB.prepare(
        "UPDATE news_items SET summary = ? WHERE id = ?"
    ).bind(summary, story.id).run();

    return { ok: true, detail: `${story.id}: summary generated` };
}
```

### 8. `generateSingleSummary` â€” article content fetching

```typescript
async function generateSingleSummary(
    story: HNStory,
    apiKey: string,
): Promise<string> {
    let articleContent = story.title;

    if (story.url) {
        try {
            const res = await fetch(story.url, {
                headers: { "User-Agent": "Mozilla/5.0" },
                signal: AbortSignal.timeout(10000),
            });
            if (res.ok) {
                const html = await res.text();
                // Strip HTML tags, get readable text (first ~8000 chars)
                const text = html.replace(/<[^>]+>/g, " ")
                    .replace(/\s+/g, " ")
                    .trim();
                // Limit to 8k chars to keep within Gemini context
                articleContent = `Title: ${story.title}\n\nArticle text:\n${text.slice(0, 8000)}`;
            }
        } catch {
            // fallback to title only
            articleContent = `Title: ${story.title}`;
        }
    }

    const prompt = `Rewrite the following article in your own words while preserving ALL content, structure, details, examples, and nuance. The output should be roughly the same length as the original. Use clear Markdown formatting with paragraphs. Do not shorten or condense â€” the goal is a full-length reformatted version.

${articleContent}`;

    const url = `https://generativelanguage.googleapis.com/v1beta/models/gemma-4-31b-it:generateContent?key=${apiKey}`;
    const res = await fetch(url, { ... });
    // parse response text
    return text;
}
```
