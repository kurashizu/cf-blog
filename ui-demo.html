Getting Started with Cloudflare Workers

May 1, 2025 · 5 min read · Kurashizu
cloudflare tutorial

Cloudflare Workers is a serverless platform that lets you run JavaScript at the edge, close to your users. In this tutorial, we will cover the basics of creating and deploying your first Worker.

Prerequisites

Before we start, make sure you have the Wrangler CLI installed and your Cloudflare account configured. You will also need Node.js 18 or later installed on your machine.

Run the following command to install Wrangler:

npm install -g wrangler

Once installed, authenticate with your Cloudflare account using wrangler login.
